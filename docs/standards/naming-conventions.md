# 文档命名规范

## 文件命名格式

### 进度文档 (`/docs/progress/`)
```
PROGRESS-{YYYYMMDD}-{简短描述}.md
```
示例：`PROGRESS-20260118-初始化项目结构.md`

### 完成报告 (`/docs/reports/completed/`)
```
COMPLETED-{YYYYMMDD}-{功能或模块名称}.md
```
示例：`COMPLETED-20260118-硬件数据库模块.md`

### 验收报告 (`/docs/reports/`)
```
REVIEW-{YYYYMMDD}-{验收内容}.md
```
示例：`REVIEW-20260118-v0.1.0发布验收.md`

## 文档内容规范

### 时间戳格式
- 日期：`YYYY-MM-DD`
- 时间：`HH:MM`
- 完整时间戳：`YYYY-MM-DD HH:MM`

### 状态标记
- `[待开始]` - 任务尚未开始
- `[进行中]` - 任务正在执行
- `[已完成]` - 任务已完成
- `[已阻塞]` - 任务被阻塞
- `[已取消]` - 任务已取消

### 优先级标记
- `[P0]` - 紧急/关键
- `[P1]` - 高优先级
- `[P2]` - 中优先级
- `[P3]` - 低优先级
